# BlurAway

BlurAway is a browser extension that hides or blurs profile pictures on LinkedIn for distraction-free browsing.

## Features
- Blurs profile images on LinkedIn feeds and comments
- Automatically blurs dynamically loaded content

## Installation

### Firefox
1. Go to `about:debugging#/runtime/this-firefox` in your Firefox browser.
2. Click "Load Temporary Add-on...".
3. Select the `manifest.json` file from the BlurAway extension folder.


## Usage
- Once loaded, visit LinkedIn and profile images will be blurred automatically.

## Uninstalling
- Remove the extension from your browser's extensions/add-ons page.
